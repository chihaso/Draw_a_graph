# Draw_a_graph
指定した関数のグラフを描画するプログラム。JS学習用。

# 使い方
1. 全てのファイルを同じディレクトリに保存
2. ブラウザでgraph.htmlを開く
3. リストボックスの中から関数を選択して描画するボタンを押すと、そのグラフが表示される

# 使用した標準外のライブラリ
* Paper.js
